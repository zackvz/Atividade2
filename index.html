Senristu Kowasugi Files: Franquia

Intro

Seja bem-vindo(a). Essa página é dedicada à franquia Senristu Kowasugi Files, uma série de filmes de terror japoneses notáveis por sua simplicidade nos métodos de gravação e pela atmosfera assustadora que conseguem criar. A qual sou um grande fã.

Filmes da franquia

  • Senritsu Kaiki File Kowasugi! File 01: Operation Capture the Slit-Mouthed Woman (2012)

    Primeira entrada no "mockumentário"(subgênero de filmes de terror que mistura elementos de documentário e ficção) do diretor Koji Shiraishi. Aborda sobre uma equipe de TV que investiga a lenda da "Kuchisake-onna", uma mulher com a boca cortada de orelha a orelha.

  • Senritsu Kaiki File Kowasugi! File 02: Shivering Ghost (2012)

    3 jornalistas especializados no sobrenatural seguem os passos de um fantasma, despertado por 4 amigos explorando uma escola abandonada.

  • Senritsu Kaiki File Kowasugi! File 03: Legend of a Human-Eating Kappa (2013)

    Uma nova fita foi entregue para Ichikawa. Um casal pescando num lago gravou uma estranha criatura que emergiu das águas.

  • Senritsu Kaiki File Kowasugi! File 04: The Truth! Hanako-san in the Toilet (2013)

    Uma fita recentemente enviada por duas adolescentes revela algo saindo de um banheiro escolar fechado numa escola abandonada.

  • Senritsu Kaiki File Kowasugi! Preface: True Story of the Ghost of Yotsuya (2014)

    Direto do prévio trabalho do time, diretor Kudo agora está convencido da existência de universos paralelos. Mas agora, uma nova fita mandada convence o grupo a descobrir a verdade sobre uma mulher misteriosa com o rosto deformado.

  • Senritsu Kaiki File Kowasugi! The Most Terrifying Movie in History (2014)

    Nessa sexta obra, diretor Kudo se alia com o poderoso psíquico Dogen, físico Masahiko Saito, e a idol Akari, para fazer um grande filme enquanto investiga a abandonada "Vila Tatari", onde qualquer um que nela entra fica insano ou morre de maneiras misteriosas.

  • Senritsu Kaiki File Kowasugi! File 07: The Final Chapter (2015)

    Dezoito meses se passaram e Kudo e Ichikawa continuam perdidos num universo paralelo. No momento que desapareceram, um gigante apareceu no céu de Shinjuku.
